py2http.tests.objects_for_testing.sum_of_args(*args, **kwargs)[source]¶

Sums the args, and returns a copy of kwargs with the sum of args added to each value.

>>> sum_of_args(1,2,3, a=0, b=10)
{'a': 6, 'b': 16}
