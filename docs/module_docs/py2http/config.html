py2http.config.mk_config(key, func, configs, defaults, **options)[source]¶

Get a config value for a function. First checks the properties of the function, then the configs, then the defaults.

Parameters

  • key – The key to search for

  • func – The function associated with the config

  • configs – A config dict to search

  • defaults – The default configs to fall back on

  • **options –

    Additional options

Keyword Arguments

  • funcname The name of the function, if not the same as func.__name__

  • type The expected type of the output (use Callable for functions)
